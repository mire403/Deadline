# Deadline
Deadline/截止日是一个帮你“抓住说过要做、却容易忘记的事”的AI工具。它不会生成新任务，只会在你已有的聊天、邮件、Issue评论等文本里，识别出带有承诺意味的句子，比如「我下周修一下」「我稍后跟进」「下个版本再看」等，并把它们整理成一份结构化的承诺清单。 对于每一条被识别的承诺，Deadline会记录：承诺内容、谁说的（不清楚就标记为Unassigned）、何时说的、是否有原始的截止时间表达（如 “by Friday”“next week”，没有就标记为No explicit deadline），以及一个简单的状态（pending/overdue/unclear）。所有智能识别都在单句级别完成，并通过严格的Prompt约束避免幻觉，确保只提取真实存在的承诺，而不是凭空“脑补任务”。
